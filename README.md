## Pokemon_typing (ポケモンタイピング)

## Project概要
従来のタイピングサイトでは、語源が日常で使用する言葉遊びだと感じたのでポケモンの名前でタイピングできるサイトを作りました。
Pokemon APIを使用して英語ではありますが、ゲームタイトルを入力するとそのタイトルに出現するポケモンネームがランダムで生成します。
※API使用を復習するProjectなので、サイトUI, タイム設定 , ランキング設定は無し。
